**Part A: Create `src/renderer/building.ts`**

Building class extending Container. Holds a Sprite from the building atlas and a BitmapText label. Provides agent positioning methods (idle fan-out, work position, entrance).

```typescript
import { Container, Sprite, BitmapText, Texture } from 'pixi.js';
import type { BuildingType } from '../shared/constants';
import { BUILDING_LABELS } from '../shared/constants';
```

Key design decisions:
- Sprite anchor at (0.5, 1.0) -- bottom-center for ground placement. The building's Container position is the ground-center point.
- Label displayed above building using BitmapText with 'PixelSignpost' font (same as existing HQ and Compound labels)
- `getIdlePosition(index, total)` fans agents horizontally below the building (y offset of +20 from container origin, which is below the sprite since anchor is bottom-center)
- `getWorkPosition(index, total)` fans agents horizontally near the building center (y offset of -16, partway up the building)
- `getEntrancePosition()` returns {x: 0, y: 10} -- at the building base, slightly below

Constructor signature: `constructor(buildingType: BuildingType, texture: Texture)`

The Building class does NOT need to know about the old compound system. It provides local-coordinate positions; world.ts converts to global.

```typescript
private getBuildingEntrance(building: Building): { x: number; y: number } {
  const local = building.getEntrancePosition();
  return { x: building.x + local.x, y: building.y + local.y };
}

private getBuildingWorkPosition(building: Building, sessionId: string): { x: number; y: number } {
  // Count agents already at this building to fan them out
  let agentIndex = 0;
  let totalAtBuilding = 0;
  for (const [sid, agent] of this.agents) {
    const state = agent.getState();
    if (state === 'working' || state === 'walking_to_sublocation') {
      // Check if this agent's last activity maps to the same building
      const activity = this.lastActivity.get(sid);
      if (activity && this.questZones.get(activity) === building) {
        totalAtBuilding++;
        if (sid === sessionId) agentIndex = totalAtBuilding - 1;
      }
    }
  }
  const local = building.getWorkPosition(agentIndex, Math.max(totalAtBuilding, 1));
  return { x: building.x + local.x, y: building.y + local.y };
}
```

Replace `getGlobalHQIdlePosition()` with `getGuildHallIdlePosition()` using the same fan-out logic but against guildHall instead of hq:
```typescript
private getGuildHallIdlePosition(sessionId: string): { x: number; y: number } {
  let idleCount = 1;
  let thisIndex = 0;
  for (const agent of this.agents.values()) {
    if (agent.sessionId === sessionId) continue;
    if (agent.getState() === 'idle_at_hq') {
      idleCount++;
      thisIndex++;
    }
  }
  const localPos = this.guildHall.getIdlePosition(thisIndex, idleCount);
  return {
    x: this.guildHall.x + localPos.x,
    y: this.guildHall.y + localPos.y,
  };
}
```
