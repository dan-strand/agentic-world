```typescript
export class DashboardPanel {
  private container: HTMLElement;
  private totalsBar: HTMLElement;
  private sessionList: HTMLElement;
  private expandedSessions: Set<string> = new Set();  // Track which sessions are expanded

  constructor(container: HTMLElement) {
    this.container = container;
    this.container.innerHTML = '';  // Remove placeholder text

    this.totalsBar = document.createElement('div');
    this.totalsBar.className = 'dashboard-totals';
    this.container.appendChild(this.totalsBar);

    this.sessionList = document.createElement('div');
    this.sessionList.className = 'session-list';
    this.container.appendChild(this.sessionList);
  }

  update(data: DashboardData): void {
    this.renderTotals(data.todayTotals);
    this.renderSessions(data.sessions);
  }
  // ... private methods below
}
```

**Formatting helpers** (private static methods or module-level functions):

1. `formatTokens(n: number): string` -- abbreviate large numbers:
   - `>= 1_000_000`: `(n / 1_000_000).toFixed(1) + 'M'`
   - `>= 1_000`: `Math.round(n / 1_000) + 'K'`
   - Otherwise: `n.toLocaleString()`

2. `formatCost(usd: number, isEstimate: boolean): string`:
   - Returns `~$${usd.toFixed(2)}` if not estimate
   - Returns `~$${usd.toFixed(2)} est.` if estimate

3. `formatDuration(lastModified: number): string`:
   - `const mins = Math.floor((Date.now() - lastModified) / 60_000)`
   - If `mins < 1`: `'<1m'`
   - If `mins >= 60`: `Math.floor(mins / 60) + 'h ' + (mins % 60) + 'm'`
   - Otherwise: `mins + 'm'`

**renderTotals(totals: TodayTotals):** Replace innerHTML of `this.totalsBar`:

**renderSessions(sessions: DashboardSession[]):** Clear and rebuild `this.sessionList`. For each session, create a row div:

```html

    <span class="project-name">{projectName}</span>
    <span class="status-badge {status}">{status}</span>
    <span class="model-badge">{modelDisplayName}</span>
    <span class="tool-name">{lastToolName || '--'}</span>
    <span class="duration">{formatDuration(lastModified)}</span>
    <span class="cost">{formatCost(totalCostUsd, isEstimate)}</span>
  </div>
  <div class="session-detail" style="display: {expanded ? 'block' : 'none'};">
    <div class="token-row"><span class="token-label">Input:</span> <span class="token-value">{inputTokens.toLocaleString()}</span></div>
    <div class="token-row"><span class="token-label">Output:</span> <span class="token-value">{outputTokens.toLocaleString()}</span></div>
    <div class="token-row"><span class="token-label">Cache write:</span> <span class="token-value">{cacheCreationTokens.toLocaleString()}</span></div>
    <div class="token-row"><span class="token-label">Cache read:</span> <span class="token-value">{cacheReadTokens.toLocaleString()}</span></div>
    <div class="token-row cost-row"><span class="token-label">Cost:</span> <span class="token-value">{formatCost(totalCostUsd, isEstimate)}</span></div>
```

**Click handler:** Attach a click listener to each `.session-summary` div that toggles the session in `this.expandedSessions` set and shows/hides the `.session-detail` div. Preserve expanded state across updates (don't collapse sessions when new data arrives).

**Sort sessions:** Sort by status priority (active first, then waiting, then idle), then by lastModified descending within each status group.

**Empty state:** If `sessions.length === 0`, show a muted message: "No active sessions" in the session list area.
