**src/renderer/bitmap-font.ts** -- Pixel art font initialization.

Per research Pattern 5: use BitmapFont.install() to create a pixel-art style font programmatically at init time.

```typescript
import { BitmapFont } from 'pixi.js';

export function installPixelFont(): void {
  BitmapFont.install({
    name: 'PixelSignpost',
    style: {
      fontFamily: 'monospace',
      fontSize: 16,
      fill: '#ffffff',
    },
    chars: [
      ['a', 'z'],
      ['A', 'Z'],
      ['0', '9'],
      [' ', ' '],
      ['-', '-'],
      ['.', '.'],
      ['_', '_'],
      ['/', '/'],
      ['\\', '\\'],
    ],
    textureStyle: {
      scaleMode: 'nearest',  // crisp pixels, no interpolation
    },
  });
}
```

Export only `installPixelFont`. Call once at world init before any BitmapText is created.

**src/renderer/activity-icons.ts** -- Small GraphicsContext icons for speech bubbles.

Create 12-16px icons drawn with Graphics primitives. Each icon represents an activity type:
- **coding** (wrench icon): Two diagonal lines forming an X-wrench shape. Color: white or light gray.
- **reading** (magnifying glass): Circle with a short line handle. Color: white or light gray.
- **testing** (gear icon): Circle with small rectangles radiating outward (cog teeth). Color: white or light gray.
- **comms** (antenna icon): Triangle/cone shape with radiating arc lines. Color: white or light gray.
- **idle** (zzz icon or pause bars): Two vertical parallel bars or "ZZ" shape.

Each is a `GraphicsContext` object pre-built at init time.

```typescript
import { GraphicsContext } from 'pixi.js';
import { ActivityType } from '../shared/types';

const iconContexts = new Map<ActivityType, GraphicsContext>();

export function initActivityIcons(): void {
  // Build each icon as a GraphicsContext
  // coding icon -- wrench
  const coding = new GraphicsContext();
  // ... draw wrench with .rect(), .moveTo(), .lineTo(), .stroke()
  iconContexts.set('coding', coding);
  // ... repeat for reading, testing, comms, idle
}

export function getActivityIcon(activity: ActivityType): GraphicsContext | undefined {
  return iconContexts.get(activity);
}
```

Keep icons simple -- 3-5 Graphics primitives each. They render at ~14x14 px inside the speech bubble.

**src/renderer/speech-bubble.ts** -- Per research Pattern 6.

```typescript
import { Container, Graphics, GraphicsContext } from 'pixi.js';
import { ActivityType } from '../shared/types';
import { SPEECH_BUBBLE_DURATION, SPEECH_BUBBLE_FADE_MS } from '../shared/constants';
import { getActivityIcon } from './activity-icons';

export class SpeechBubble extends Container {
  private bubble: Graphics;     // White rounded rect background
  private icon: Graphics;       // Activity icon (context-swapped)
  private fadeTimer = 0;
  private isActive = false;

  constructor() {
    super();
    // Draw bubble background: white rounded rect ~28x24, with small triangle pointer at bottom
    this.bubble = new Graphics();
    this.bubble.roundRect(0, 0, 28, 24, 4).fill(0xffffff);
    // Triangle pointer
    this.bubble.moveTo(10, 24).lineTo(14, 30).lineTo(18, 24).fill(0xffffff);
    this.addChild(this.bubble);

    // Icon placeholder (context swapped on show)
    this.icon = new Graphics();
    this.icon.position.set(7, 5);  // Centered in bubble
    this.addChild(this.icon);

    // Start hidden
    this.visible = false;

    // Position above agent head
    this.position.set(-14, -60);
  }

  show(activity: ActivityType): void {
    const iconCtx = getActivityIcon(activity);
    if (iconCtx) {
      this.icon.context = iconCtx;
    }
    this.alpha = 1;
    this.visible = true;
    this.fadeTimer = 0;
    this.isActive = true;
  }

  tick(deltaMs: number): void {
    if (!this.isActive) return;
    this.fadeTimer += deltaMs;
    if (this.fadeTimer > SPEECH_BUBBLE_DURATION) {
      const fadeProgress = (this.fadeTimer - SPEECH_BUBBLE_DURATION) / SPEECH_BUBBLE_FADE_MS;
      this.alpha = Math.max(0, 1 - fadeProgress);
      if (this.alpha <= 0) {
        this.visible = false;
        this.isActive = false;
      }
    }
  }
}
```

Per user decision: bubbles use minimal icons (not text), flash on activity change (3-5 sec then fade), are NOT persistent.
