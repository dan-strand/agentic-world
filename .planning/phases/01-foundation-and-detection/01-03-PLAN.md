**src/renderer/game-loop.ts** -- Adaptive ticker management:

Create class `GameLoop`:
```typescript
export class GameLoop {
  private app: Application;
  private world: World;
  private isIdle: boolean;
  private isMinimized: boolean;

  constructor(app: Application, world: World);
  start(): void;
  onSessionsUpdate(sessions: SessionInfo[]): void;
  onWindowMinimized(): void;
  onWindowRestored(): void;
  stop(): void;
}
```

`constructor`:
- Store references to app and world
- Set initial state: isIdle = true, isMinimized = false

`start()`:
- Set `app.ticker.maxFPS = FPS_IDLE` (5 fps -- start conservatively)
- Add ticker callback: `app.ticker.add((ticker) => { this.world.tick(ticker.deltaMS); })`
- Start the ticker

`onSessionsUpdate(sessions)`:
- Check if any session has status === 'active'
- If active sessions exist AND currently idle:
  - Set `app.ticker.maxFPS = FPS_ACTIVE` (30 fps)
  - If ticker was stopped, call `app.ticker.start()`
  - Set isIdle = false
- If NO active sessions AND currently not idle:
  - Set `app.ticker.maxFPS = FPS_IDLE` (5 fps)
  - Set isIdle = true
- If NO sessions at all and ticker is running, consider stopping ticker entirely (0 CPU) and only restart on next session update

`onWindowMinimized()`:
- Call `app.ticker.stop()` -- stop rendering entirely
- Set isMinimized = true

`onWindowRestored()`:
- Call `app.ticker.start()` -- resume rendering
- Set isMinimized = false
- Restore appropriate FPS based on isIdle state

`stop()`:
- Remove ticker callback
- Stop ticker

**src/renderer/index.ts** -- Complete wiring (REPLACE the stub from Plan 01):

```typescript
import { World } from './world';
import { GameLoop } from './game-loop';
import { SessionInfo } from '../shared/types';

async function main(): Promise<void> {
  // 1. Init PixiJS world
  const appContainer = document.getElementById('app')!;
  const world = new World();
  await world.init(appContainer);

  // 2. Init adaptive game loop
  const gameLoop = new GameLoop(world.getApp(), world);
  gameLoop.start();

  // 3. Wire IPC -> visuals
  window.agentWorld.onSessionsUpdate((sessions: SessionInfo[]) => {
    world.updateSessions(sessions);
    gameLoop.onSessionsUpdate(sessions);
  });

  // 4. Load initial sessions
  const initialSessions = await window.agentWorld.getInitialSessions();
  world.updateSessions(initialSessions);
  gameLoop.onSessionsUpdate(initialSessions);

  // 5. Handle window resize
  window.addEventListener('resize', () => {
    world.resize();
  });

  // 6. Handle minimize/restore for adaptive frame rate
  // Electron fires blur/focus on minimize/restore in some cases
  // Use document.visibilitychange for reliable minimize detection
  document.addEventListener('visibilitychange', () => {
    if (document.hidden) {
      gameLoop.onWindowMinimized();
    } else {
      gameLoop.onWindowRestored();
    }
  });

  console.log('Agent World initialized');
}

main().catch(err => {
  console.error('Failed to initialize Agent World:', err);
});
```
