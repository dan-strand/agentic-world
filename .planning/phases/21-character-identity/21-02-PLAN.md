**Part A: Create `src/renderer/palette-swap.ts`** — Runtime palette swap utility.

This module takes the base character animation textures and produces palette-swapped copies for a specific agent's palette. Uses an offscreen canvas to read/modify pixel data.

```typescript
import { Texture, ImageSource } from 'pixi.js';
import type { CharacterClass } from '../shared/types';
import { PALETTE_DEFS, TEMPLATE_COLORS } from '../shared/constants';
import type { PaletteDef } from '../shared/constants';

/**
 * Cache of palette-swapped texture arrays.
 * Key: `${characterClass}_${paletteIndex}_${animState}` -> Texture[]
 * Prevents re-creating textures for same class+palette combo.
 */
const swapCache: Map<string, Texture[]> = new Map();

/**
 * Create palette-swapped animation textures for a specific character class and palette.
 * Takes the base animation textures and replaces template colors with palette colors.
 *
 * @param baseTextures - Original animation texture array from characterAnimations
 * @param characterClass - The character class (determines which template colors to replace)
 * @param paletteIndex - Index into PALETTE_DEFS
 * @returns New Texture[] with palette-swapped colors
 */
export function createPaletteSwappedTextures(
  baseTextures: Texture[],
  characterClass: CharacterClass,
  paletteIndex: number,
): Texture[] { ... }
```

Implementation approach:
1. Build cache key from class + palette index + first texture UID (to distinguish anim states)
2. Return from cache if already created
3. For each texture in baseTextures:
   a. Create an offscreen canvas (32x32)
   b. Draw the base texture to the canvas
   c. Get ImageData pixels
   d. For each pixel, check if it matches a template color (within tolerance of +/-15 per channel to catch shading variants)
   e. If match found, replace with the corresponding palette color, preserving the brightness delta (so shaded variants of template colors get correspondingly shaded palette colors)
   f. Put modified ImageData back
   g. Create a new Texture from the canvas
4. Store in cache and return

**Color matching with brightness preservation:**
For each pixel, compute the channel-wise delta from the template color. If the absolute delta per channel is within tolerance (15), apply the replacement:
```
newR = palette.robe[0] + (pixel.r - template.robe[0])
newG = palette.robe[1] + (pixel.g - template.robe[1])
newB = palette.robe[2] + (pixel.b - template.robe[2])
```
Clamp to 0-255. This preserves shading/highlight detail while changing the base hue.

Check all three template regions (robe, hair, accent) for each pixel. Use the TEMPLATE_COLORS for the given characterClass.
