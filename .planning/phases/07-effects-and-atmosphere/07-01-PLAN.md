1. Create `src/renderer/level-up-effect.ts`:

   Import `Container`, `Graphics`, `FillGradient` from `pixi.js`. Import `GlowFilter` from `pixi-filters`. Import LEVEL_UP_* constants from `../shared/constants`.

   Class `LevelUpEffect extends Container`:
   - Private fields: `sparkles: Sparkle[]`, `column: Graphics`, `elapsed: number`, `done: boolean`, `columnAlpha: number`
   - Interface `Sparkle { gfx: Graphics; vx: number; vy: number; life: number; maxLife: number; }`
   - Constructor(x, y):
     - `super(); this.position.set(x, y);`
     - Create light column: `new Graphics()` with `FillGradient` vertical gradient from transparent at top to gold (rgba(255, 215, 0, 0.9)) at bottom. The column is LEVEL_UP_COLUMN_WIDTH wide and LEVEL_UP_COLUMN_HEIGHT tall, extending upward from origin (y: -LEVEL_UP_COLUMN_HEIGHT to 0). Use `FillGradient({ type: 'linear', start: { x: 0, y: -LEVEL_UP_COLUMN_HEIGHT }, end: { x: 0, y: 0 }, colorStops: [...] })`.

     **IMPORTANT FillGradient fallback:** If FillGradient with rgba color stops does not render alpha correctly (this is an open question from research), use a fallback approach: draw 5-6 overlapping rectangles with decreasing alpha values to simulate the gradient. Test the FillGradient approach first.

     - Create sparkles: Loop LEVEL_UP_SPARKLE_COUNT times. Each sparkle is a small Graphics circle (radius 1.5 + random * 1.5), filled with a random color from LEVEL_UP_SPARKLE_COLORS. Initial position at (0, 0). Random velocity: vx = (random - 0.5) * 60, vy = -20 + random * 40 (mostly downward shower). Life starts at 0, maxLife = 1200 + random * 800.
     - Apply GlowFilter: `this.filters = [new GlowFilter({ distance: LEVEL_UP_GLOW_DISTANCE, outerStrength: LEVEL_UP_GLOW_OUTER_STRENGTH, innerStrength: 0.5, color: LEVEL_UP_GLOW_COLOR, quality: LEVEL_UP_GLOW_QUALITY })]`

   - `tick(deltaMs)`:
     - If done, return early.
     - Increment elapsed by deltaMs.
     - Update column alpha: fade out in the last 30% of duration. `columnAlpha = elapsed < LEVEL_UP_DURATION_MS * 0.7 ? 1 : 1 - (elapsed - LEVEL_UP_DURATION_MS * 0.7) / (LEVEL_UP_DURATION_MS * 0.3)`. Apply to column: `this.column.alpha = Math.max(0, columnAlpha)`.
     - Update sparkles: for each sparkle, increment life by deltaMs, apply gravity (vy += 40 * dt), update position (gfx.x += vx * dt, gfx.y += vy * dt), fade alpha based on life/maxLife ratio. Mark sparkle gfx invisible when life > maxLife.
     - Mark done when elapsed >= LEVEL_UP_DURATION_MS.

   - `isDone(): boolean` -- return this.done.

   Export the class as a named export.

2. Create `src/renderer/ambient-particles.ts`:

   Import `Container`, `Graphics` from `pixi.js`. Import AMBIENT_* constants and WORLD_WIDTH, WORLD_HEIGHT from `../shared/constants`.

   Interface `AmbientParticle { gfx: Graphics; baseY: number; driftSpeed: number; bobSpeed: number; bobAmplitude: number; alphaSpeed: number; phase: number; }`

   Class `AmbientParticles extends Container`:
   - Private field: `particles: AmbientParticle[]`
   - Constructor():
     - `super();`
     - Loop AMBIENT_PARTICLE_COUNT times:
       - Create `new Graphics()`, draw circle at (0,0) with radius (AMBIENT_PARTICLE_SIZE_MIN + random * (SIZE_MAX - SIZE_MIN)), fill with AMBIENT_PARTICLE_COLOR.
       - Random initial position: x = random * WORLD_WIDTH, y = random * WORLD_HEIGHT.
       - Random initial alpha for staggered effect.
       - Push to particles array with randomized driftSpeed (DRIFT_MIN + random * (DRIFT_MAX - DRIFT_MIN)), bobSpeed (0.5 + random), bobAmplitude (BOB_AMP_MIN + random * (BOB_AMP_MAX - BOB_AMP_MIN)), alphaSpeed (0.3 + random * 0.5), phase (random * PI * 2).
       - addChild(gfx).

   - `tick(deltaMs)`:
     - `const dt = deltaMs / 1000;`
     - For each particle:
       - Increment phase by dt.
       - Update x: `gfx.x += driftSpeed * dt`
       - Update y: `gfx.y = baseY + Math.sin(phase * bobSpeed) * bobAmplitude`
       - Update alpha: `gfx.alpha = AMBIENT_PARTICLE_ALPHA_MIN + AMBIENT_PARTICLE_ALPHA_RANGE * ((Math.sin(phase * alphaSpeed) + 1) / 2)`
       - Wrap: if `gfx.x > WORLD_WIDTH + 10`, reset to `gfx.x = -10` and randomize baseY.

   Export the class as a named export.

   **Key anti-pattern avoidance:**
   - Do NOT use ParticleContainer (overkill for 25 particles, and it doesn't support filters).
   - Particles wrap to x = -10 (offscreen left) to avoid pop-in artifact.
