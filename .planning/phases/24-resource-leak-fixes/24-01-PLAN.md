<objective>
Eliminate GPU resource leaks from particle systems, palette swap texture cache, and GlowFilter shaders.

Purpose: These three leak categories (LEAK-01, LEAK-02, LEAK-03) account for the majority of unbounded GPU memory growth during long runs. Particle create/destroy churn causes ~3,360 Graphics allocations per hour. Palette swap textures accumulate without cleanup. GlowFilter shaders leak on every celebration.

Output: GraphicsPool utility, refactored particle systems, palette swap cache lifecycle, filter cleanup -- all with unit tests.
</objective>
