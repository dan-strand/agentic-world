**In `src/shared/constants.ts`:**
Add smoke particle constants:
```typescript
// Chimney smoke particle constants (Phase 20)
export const CHIMNEY_SMOKE_COUNT = 5;           // Max simultaneous smoke particles per building
export const CHIMNEY_SMOKE_SPAWN_MS = 800;      // Time between new smoke particle spawns
export const CHIMNEY_SMOKE_RISE_SPEED = 12;     // Pixels per second upward drift
export const CHIMNEY_SMOKE_DRIFT_SPEED = 4;     // Pixels per second horizontal drift
export const CHIMNEY_SMOKE_LIFETIME_MS = 3000;  // How long each puff lives
export const CHIMNEY_SMOKE_SIZE_MIN = 2;        // Starting radius of smoke puff
export const CHIMNEY_SMOKE_SIZE_MAX = 5;        // Max radius as smoke puff grows
export const CHIMNEY_SMOKE_COLOR = 0x999999;    // Gray smoke color
```

Also add a chimney position offset constant per building type (the chimney position relative to building origin, which is at bottom-center due to anchor 0.5, 1.0):
```typescript
export const CHIMNEY_POSITIONS: Record<BuildingType, { x: number; y: number }> = {
  wizard_tower:     { x: -100, y: -330 },  // Top-left area of building
  training_grounds: { x: 120,  y: -330 },  // Top-right area
  ancient_library:  { x: -80,  y: -330 },  // Top-left area
  tavern:           { x: 60,   y: -332 },  // Near center, slightly higher (larger chimney)
  campfire:         { x: 0, y: 0 },        // Not used
};
```
Adjust the Y values to be approximately `-(BUILDING_HEIGHT - 6)` since the chimney is at the top of the building, and X values should match where the chimney was drawn in the building atlas. These are estimates -- the executor should cross-reference with the chimney positions drawn in Task 1.

**In `src/renderer/building.ts`:**

Add a private smoke particle system to the Building class. The system should:

1. **Smoke particle interface:** `{ gfx: Graphics, age: number, lifetime: number, vx: number, vy: number }`

2. **Private fields:** Add `smokeParticles: SmokeParticle[]`, `smokeTimer: number`, `smokeContainer: Container` to Building.

3. **Constructor:** Create a `smokeContainer` Container and add it as a child of the Building (after all existing children, so smoke renders on top). Skip smoke for campfire building type.

4. **`tick(deltaMs)` method:** Add a new public `tick(deltaMs: number)` method to Building:
   - Advance smokeTimer. When it exceeds CHIMNEY_SMOKE_SPAWN_MS, spawn a new smoke particle:
     - Create a small Graphics circle (CHIMNEY_SMOKE_SIZE_MIN radius, CHIMNEY_SMOKE_COLOR, alpha 0.6)
     - Position at the chimney location (from CHIMNEY_POSITIONS)
     - Give random slight horizontal drift (CHIMNEY_SMOKE_DRIFT_SPEED * random direction)
     - Add to smokeContainer and smokeParticles array
   - Tick each existing particle:
     - Move upward (vy = -CHIMNEY_SMOKE_RISE_SPEED * dt)
     - Move horizontally (vx * dt)
     - Increase age
     - Scale up slightly as smoke expands (lerp radius from MIN to MAX over lifetime)
     - Fade alpha toward 0 as age approaches lifetime
     - Remove particles that exceed CHIMNEY_SMOKE_LIFETIME_MS (destroy Graphics, remove from array)
   - Cap total particles at CHIMNEY_SMOKE_COUNT (don't spawn if at max)

5. **Performance:** This is lightweight -- at most 5 small Graphics circles per building (20 total for 4 buildings). No filters or complex shaders needed.

The Building class currently has no tick() method, so world.ts will need to call it. Add a comment noting that world.ts should call `building.tick(deltaMs)` in its tick loop -- this will be wired in Plan 20-03.
