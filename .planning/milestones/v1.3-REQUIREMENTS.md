### Audio -- Ready to Work

- [x] **AUDIO-04**: "Ready to work" reminder timer starts from "waiting" status (not "idle")
- [x] **AUDIO-05**: Each session gets its own independent 1-minute waiting timer
- [x] **AUDIO-06**: Reminder sounds are throttled with ~30s minimum gap between any two reminder plays
- [x] **AUDIO-07**: Reminder does not repeat for a session until that session returns to "active" and goes back to "waiting"

### Status Reliability

- [x] **STATUS-01**: Session status transitions (active->waiting->idle) are accurately detected from JSONL files
- [x] **STATUS-02**: Status debounce correctly commits transitions without dropping or duplicating state changes
- [x] **STATUS-03**: Sessions that reactivate after being dismissed (faded out) properly reappear as agents

### Session Visibility

- [x] **VIS-01**: All active/waiting sessions always have a visible agent on screen (no invisible or missing agents)
- [x] **VIS-02**: Active sessions with failed tool detection default to a building (not guild hall)
- [x] **VIS-03**: Stale session filter (30 min) does not incorrectly remove actively-working sessions
